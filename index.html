Petri Dish

an experiment with animated canvas

  • This is a page to demonstrate Gyudon.js, a Javascript library that I wrote, and what it can be used to create. Gyudon.js wraps around the HTML Canvas API and allows for construction of 2D shapes on the canvas while also allowing for easy animation and manipulation of those shapes.
  • Javascript must be enabled to view this page as intended. To view the different examples, just select a pattern on the left menu and either 'Wireframe' or 'Filled' from under the canvas. If you experience a problem viewing the page, do not hesitate to report it.
  • I've added this demonstration to Github.
